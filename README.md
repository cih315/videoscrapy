django-admin-bootstrapped
=========================

国内视爬虫

## Requirements

* Django `1.4.x`.
* Scrapy `Scrapy 0.16`.

## Installation

1. Download it from PyPi with `pip install django-admin-bootstrapped`
2. Add `'django_admin_bootstrapped'` into the `INSTALLED_APPS` list __before__ `'django.contrib.admin'`
3. Have fun!
